My Choose-Your-Own Adventure Project

Choose-Your-Own Adventure Project Link

The purpose of this project is to create an 8-page “Choose Your Own Adventure” website that demonstrates my understanding of HTML structure, navigation, and storytelling. The objective is to use correct file organization and include required elements such as paragraph tags, headings, inline formatting tags, at least one image, and a properly formatted list. The site must be coherent, entertaining, easy to navigate, logically organized, and the code should be clean and commented where ideas or code were borrowed.

My Business Card Project

Business Card Project Link

The primary purpose of this project is to demonstrate mastery of fundamental CSS layout techniques by creating two distinct business cards. One using only Box Model properties and the other using Flexbox; while also applying key design principles such as Contrast, Repetition, Alignment, and Proximity.
