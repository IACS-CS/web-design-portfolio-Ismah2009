My Choose-Your-Own Adventure Project

Choose-Your-Own Adventure Project Link

The purpose of this project is to create an 8-page “Choose Your Own Adventure” website that demonstrates my understanding of HTML structure, navigation, and storytelling. The objective is to use correct file organization and include required elements such as paragraph tags, headings, inline formatting tags, at least one image, and a properly formatted list. The site must be coherent, entertaining, easy to navigate, logically organized, and the code should be clean and commented where ideas or code were borrowed.

My Business Card Project

Business Card Project Link

The purpose of this project is to demonstrate mastery of fundamental CSS layout techniques by creating two distinct business cards. One using only Box Model properties and the other using Flexbox; while also applying key design principles such as Contrast, Repetition, Alignment, and Proximity.

My CSS Zen Garden Project

CSS Zen Garden Project Link

The purpose of this project is to demonstrate the power and flexibility of CSS by transforming a fixed HTML structure into a visually unique and engaging design using only a custom stylesheet. This project highlights my ability to apply advanced CSS techniques; such as layout systems, typography, color theory, and responsiveness; while following core design principles including Contrast, Repetition, Alignment, and Proximity. Multiple designs showcase different layout approaches and creative experimentation, emphasizing how CSS alone can significantly change the look and feel of a webpage.

My Interactive Design Project

Interactive Design Project Link

The purpose of this project is to demonstrate the integration of visual design and user interaction by creating a single-page, interactive explainer website using HTML, CSS, and vanilla JavaScript. The project focuses on guiding users through information using reveals, content changes, and visual feedback, while applying core design principles such as Contrast, Repetition, Alignment, and Proximity. Through interactive elements and thoughtful layout choices, this project highlights how motion and user input can enhance understanding and engagement.

My Design Improvement Project

Design Improvement Project Link

The purpose of this project is to demonstrate the application of core graphic and web design principles by analyzing an existing design and recreating it with intentional improvements using HTML and CSS. The project focuses on identifying visual flaws and enhancing hierarchy, readability, and cohesion through effective use of Contrast, Repetition, Alignment, and Proximity, as well as thoughtful color and typography choices.
